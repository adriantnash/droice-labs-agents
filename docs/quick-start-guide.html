Background Check Agent — Quick Start Guide

Use this guide to run background checks on people you are meeting with. No technical knowledge required.

Step 1

Open the agent

Double-click launch.bat in the shared folder. A window will open and the agent will start automatically. Wait for it to say "Ready. Type your request below."

Step 2

Type your request

Use one of the templates below, fill in the blanks, and press Enter. You can include as little or as much detail as you have — at minimum, a name and company are enough.

Step 3

Wait for the report

The agent will research the person and build the report automatically. This takes 2–5 minutes per person. Do not close the window while it is working.

Step 4

Find your file

When finished, the agent will tell you the file name. Open the reports folder in the shared drive to find your Word document.

Copy & Paste Templates

Single Person

Run a background check on [Full Name], [Job Title] at [Company]. Meeting date: [Date] LinkedIn: [URL — optional]

Multiple People

Run background checks on everyone in this list: 1. [Full Name], [Title], [Company] 2. [Full Name], [Title], [Company] 3. [Full Name], [Title], [Company] Meeting date: [Date]

Quick / Minimal Info

Background check on [Full Name] from [Company].

From a Spreadsheet / List

Run background checks on everyone in this list: [Paste your rows here directly from Excel or a CSV file]

Where is my report?

Reports are saved as Word documents (.docx) in the reports folder. The agent will tell you the exact file name when it finishes. Open the file in Microsoft Word as you normally would.

Tips

🔑

You do not need a LinkedIn URL. A name and company are enough. LinkedIn URLs improve results if you have them, but are optional.

🕐

Reports take 2–5 minutes per person. For a list of 5 people, expect around 15–20 minutes. You can leave the window running in the background.

📄

Multiple people from the same company are combined into one report automatically — one file covers everyone at that company.

⚠

Do not close the window while the agent is working. If the window closes unexpectedly, re-open launch.bat and try again.

🔄

The agent updates automatically each time you open it, so you always have the latest version with no action required from you.
